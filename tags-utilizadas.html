Lista de Tags Utilizadas

  • !DOCTYPE html - Declara o tipo de documento como HTML5
  • html - Elemento raiz que envolve todo o conteúdo da página
  • head - Contém metadados e links para recursos externos
  • meta - Define metadados como charset e viewport
  • title - Define o título da página que aparece na aba do navegador
  • body - Contém o conteúdo visível da página
  • header - Define o cabeçalho da página, geralmente contém navegação
  • nav - Define uma seção de navegação com links
  • a - Define um link para outra página ou recurso
  • h1, h2 - Definem títulos e subtítulos, respectivamente
  • p - Define um parágrafo de texto
  • ul - Define uma lista não ordenada (com marcadores)
  • li - Define um item dentro de uma lista
  • link - Usado para vincular recursos externos, como folhas de estilo CSS
